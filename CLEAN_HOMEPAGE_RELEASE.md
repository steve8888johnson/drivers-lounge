# Clean Homepage Release

This release simplifies the Drivers Lounge homepage and adds the new red, white and blue interstate-style branding.

## Included
- Large classic black truck logo on the right side of the hero
- "Built by drivers for drivers" brand message
- Simplified homepage with only three primary destinations
- Cleaner navigation header
- Mobile and tablet responsive layout
- Existing navigation, road intelligence, carriers, jobs and community modules retained

## Deploy
Copy the contents of this folder into your local `Documents\\GitHub\\drivers-lounge` repository, commit, and push.

Suggested commit message:

`Add clean branded homepage`
