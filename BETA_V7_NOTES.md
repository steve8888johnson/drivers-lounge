# Drivers Lounge Production Beta v7

## Major changes
- Rebuilt homepage as a live driver operating-system dashboard
- Reduced logo dominance while retaining strong brand identity
- Added live parking, scale, fuel, weather and road-event summary cards
- Added route briefing preview and route score
- Added homepage AI Driver Copilot
- Added fast access to all major modules
- Added Business Tools foundation for owner-operators
- Improved dashboard greeting and module cohesion
- Updated manifest, sitemap and service-worker cache

## Current limitations
- Live cards use demonstration data until feeds are connected
- AI copilot uses deterministic beta responses
- Navigation is not certified commercial routing
- Business calculations are placeholders until user accounting data is connected
