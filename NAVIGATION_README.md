# Navigation-first beta

This release includes an interactive planning UI, truck profiles, hazmat and oversize modes, three route choices, route comparison, local saved trips, map layers and staged driver reporting.

It intentionally does not claim that preview routes are legally validated. Production turn-by-turn navigation requires a licensed commercial routing provider (Trimble, HERE or equivalent), background GPS, offline map packages and road testing.

Run `supabase/003_navigation_foundation.sql` after the existing Drivers Lounge database setup to add persistent truck profiles, trips and road reports.
