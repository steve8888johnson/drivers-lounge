# One Push Install

1. Extract this ZIP.
2. Open `Documents\GitHub\drivers-lounge`.
3. Delete everything inside that repository folder.
4. Copy all files and folders from this package into it.
5. In GitHub Desktop commit: `Deploy stable Drivers Lounge beta v2`.
6. Push origin.
7. Vercel will deploy automatically.

Do not copy the outer ZIP folder into the repository; copy its contents.
