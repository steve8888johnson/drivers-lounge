# One Push Update

1. Extract this ZIP.
2. Open GitHub Desktop and select `drivers-lounge`.
3. Choose Repository > Show in Explorer.
4. Replace everything inside the repository folder with everything inside this package.
5. In GitHub Desktop, commit with: `Deploy all-in-one Drivers Lounge beta`
6. Click Push origin. Vercel will deploy automatically.
7. In Supabase SQL Editor, run in order if not already run:
   - `supabase/003_navigation_foundation.sql`
   - `supabase/004_road_intelligence.sql`
